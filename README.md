# AgentForge

> Your AI team, on-demand. Deploy 120+ specialist agents in seconds.

A pixel RPG style AI Agent marketplace built with Next.js.

## Features

- **120+ Specialist Agents** - From Sales to Engineering, Marketing to Design
- **Pixel RPG Style UI** - Retro gaming aesthetic with Press Start 2P font
- **Dual Export Modes**:
  - **Agent Config** - Run as standalone agent in OpenClaw
  - **Skill Config** - Add as skill to existing agent
- **Category Filtering** - Browse agents by specialization
- **Responsive Design** - Works on mobile, tablet, and desktop

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Press Start 2P, VT323

## Data Source

All agent data comes from [agency-agents](https://github.com/msitarzewski/agency-agents) - an open-source collection of 120+ specialized AI agent personalities.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

Automatically deployed to Vercel: https://agentforge.sh
